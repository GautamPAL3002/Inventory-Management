

# 📦 SSPL Inventory Management System

A web-based inventory management application built with **Spring Boot (Java)** and a **HTML/CSS/JS frontend**, designed to efficiently manage, track, and categorize inventory items. Admin users can perform full CRUD operations, while guests have read-only access.

## 🔧 Key Features

* 🔐 **Login System** for admin access
* 📋 **All Inventory Table** with complete item details
* 👤 **Name Allocation View**
* 🧪 **Consumable / Non-Consumable Filters**
* 🔍 **Search functionality** in all views
* 📄 **Export to PDF** reports
* 📥 **Import from Excel (.xlsx)** files
* 🗃️ **H2 Database** support (easily replaceable with MySQL)

## 🛠 Tech Stack

* **Backend**: Spring Boot, Java
* **Frontend**: HTML, CSS, JavaScript
* **Database**: H2
* **Extras**: html2pdf.js, Apache POI (Excel)

## 🚀 Getting Started

1. Clone the repo
2. Run backend: `mvn spring-boot:run`
3. Open `index.html` to launch the frontend
